# 게시판 도메인 헌법

## 게시판 개념

- 게시판(Board)은 모든 사용자가 글을 공유하는 공간이다 (비로그인 조회 가능).
- 자유게시판은 주제 제한 없이 로그인한 사용자가 글을 작성할 수 있다.
- **탭 구조**: "전체" 탭(모든 게시글)과 "팔로잉" 탭(팔로우한 사람의 글)으로 구성된다.

## 핵심 개념

- **게시글(Post)**: 제목, 본문, 선택적 이미지로 구성. 작성자 닉네임과 함께 표시.
- **댓글(Comment)**: 게시글에 달리는 텍스트 응답. 작성자 닉네임 포함.
- **좋아요(Like)**: 게시글에 대한 사용자의 반응. 1인 1회 제한.
- **이미지**: 게시글당 선택적으로 1개 이미지 첨부 가능. Supabase Storage 저장.

## 책임 범위

- 게시글 작성, 조회, 삭제 (수정은 미구현)
- 댓글 작성 및 삭제 (본인 댓글만 삭제)
- 좋아요 토글
- 이미지 업로드 및 표시
- 전체/팔로잉 탭 피드 전환

## 원칙

- 게시글 목록·상세는 비로그인 사용자도 조회 가능하다.
- 게시글 작성은 로그인한 사용자만 가능하다.
- 삭제는 본인 게시글/댓글에만 허용된다.
- 이미지는 최대 5MB 제한이며 클라이언트에서 검증한다.
- `user_id`는 `public.profiles(id)`를 참조하여 PostgREST 자동 조인으로 닉네임을 가져온다.
- `/feed` 경로는 `/board`로 리다이렉트된다 (팔로잉 피드가 게시판 탭으로 통합됨).
